## causal mask

GPT 是从左到右生成 token 的，所以第 t 个 token 只能看第 t 个以及之前的 token，不能看未来 token。
Causal Mask 就是为了禁止模型偷看未来答案。

## self-attention 模型如何知道该看谁

小明 把 苹果 放进 书包，因为 他 很 饿

当模型看到 “他” 的时候，它需要知道 “他” 大概率指的是 “小明”
这就是 attention 要解决的问题：当前 token 需要从前面的 token 里找哪些信息？

## 先只回答 3 个问题

### 1）模型怎么看一句话里的“重点”？

这就是 ​**Attention**​。

### 2）模型为什么不能偷看后面的答案？

这就是 ​**Causal Mask**​。

### 3）模型为什么能一层一层越来越懂一句话？

这就是 ​**Transformer Block**​。
