# learnAIagent

这是一个面向中文学习者的 AI Agent 学习与作品集仓库。

目标：从 2026 年 6 月 10 日开始，用约 3 个月、每天 2 小时，交付具备真实使用价值的
“Garmin 表盘智能生成 Agent”，并通过完整产品实践建立 AI Agent 开发、评测、工程化和
项目负责能力。

## 从这里开始

1. 阅读 [学习导航](00-开始这里/学习导航.md)。
2. 完成 [能力自测](00-开始这里/能力自测.md)。
3. 按 [90 天路线图](01-学习计划/90天路线图.md) 学习。
4. 查看 [学习计划图](01-学习计划/学习计划图.md) 理解整体路径。
5. 每天复制 [每日笔记模板](02-学习笔记/每日笔记模板.md) 写学习记录。
6. 每周日填写 [每周复盘模板](02-学习笔记/每周复盘模板.md)。

## 核心原则

- 以作品驱动学习，不以“看完课程”为目标。
- 项目质量优先，不以“只是学习 Demo”为理由降低产品标准。
- 先理解原理，再使用框架；先做单 Agent，再考虑多 Agent。
- 每个功能都要有测试问题、失败案例和改进记录。
- 优先使用中文资料，英文术语统一记录到术语表。
- Agent 系统使用 JavaScript，不使用 TypeScript；目标产物为 Garmin Monkey C 代码。
- 前端使用 Vue 3 + Vite，后端使用 Node.js。
