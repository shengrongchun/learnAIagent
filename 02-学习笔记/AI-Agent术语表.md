# AI Agent 术语表

遇到英文不要背整段定义，先理解它在系统里负责什么。

| 英文 | 常用中文 | 简单解释 |
| --- | --- | --- |
| LLM | 大语言模型 | 根据上下文预测并生成内容的模型 |
| Token | 词元 | 模型处理文字和计算费用的基本单位 |
| Context Window | 上下文窗口 | 一次请求中模型能够处理的信息范围 |
| Prompt | 提示词 | 给模型的任务、背景、规则和示例 |
| Hallucination | 幻觉 | 模型生成看似合理但错误的内容 |
| Embedding | 向量表示 | 把内容变成可用于相似度比较的数字 |
| RAG | 检索增强生成 | 先找资料，再让模型依据资料回答 |
| Tool Calling | 工具调用 | 模型选择并请求程序执行外部能力 |
| Agent | 智能体 | 围绕目标自主选择步骤和工具的系统 |
| Workflow | 工作流 | 由开发者预先定义步骤和分支的流程 |
| Memory | 记忆 | 保存并在需要时取回历史信息 |
| MCP | 模型上下文协议 | 连接 AI 应用与工具、数据的开放协议 |
| Evaluation / Eval | 评测 | 用数据判断系统效果是否达到目标 |
| Observability | 可观测性 | 通过日志、追踪和指标理解系统运行情况 |
| Prompt Injection | 提示词注入 | 恶意内容诱导模型忽略规则或越权操作 |
| Human in the Loop | 人在回路 | 关键步骤由人确认、修正或接管 |
