# 第 1 天任务：建立正确认知

日期：2026 年 6 月 10 日  
总时长：2 小时

## 今天只解决一个问题

> AI Agent 与普通聊天机器人、固定工作流有什么区别？

## 任务

### 1. 阅读与观看，40 分钟

- 阅读阿里云百炼 Function Calling 文章，先理解模型如何使用工具。
- 观看“终于有人讲清 Agent 了”视频中的 LLM、RAG、Function Calling 和 Agent 章节。

具体链接见 [中文文章与视频学习资料：第 1 阶段](../03-资料索引/中文学习资料.md)。

### 2. 写出自己的解释，20 分钟

在今天的每日笔记中，用自己的话解释：

- 大模型是什么；
- RAG 是什么；
- 工作流是什么；
- Agent 是什么；
- Agent 为什么不一定比工作流更好。

每个概念最多写 100 字，不要复制资料原文。

### 3. 场景分析，40 分钟

围绕“Garmin 表盘智能生成 Agent”回答：

1. 用户是谁？
2. 用户现在如何完成任务？
3. 最浪费时间的步骤是什么？
4. 哪些步骤需要大模型？
5. 哪些步骤应该用固定规则？
6. Agent 可以调用哪些工具？
7. 做错会造成什么风险？

### 4. 产出第一张设计图，15 分钟

用 Mermaid 或纸笔画出：

`中文需求 -> Agent -> 领域知识/预览/代码生成/编译工具 -> Monkey C 代码`

### 5. 收尾，5 分钟

填写今天的学习笔记，并在进度表勾选 Day 1。
